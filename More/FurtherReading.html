Electricity

  • Sparkfun Electricity Tutorial: This tutorial goes through the basics of what electricity is, then links you to further lessons about electrical power and circuits.
  • All About Circuits: A series of online textbook chapters that take you through all aspects of electricity.
  • Electricity for Kids: A fun, interactive site for kids to learn the basics of electricity through games and quizzes.
  • Current Electricity: The Physics Classroom: Explains the physics behind the electricity used by engineers.
  • Siemens Basics of Electricity: An online course designed to train Siemens engineers. Siemens has many similar courses on other aspects of electrical engineering as well.
  • Leonics Understanding Electricity: A renewable energy company explains electricity from an industry perspective.
  • Electricity Teaching Resources: Games and interactive tutorials to help kids learn about circuits.

Software

These resources teach coding in general. Arduino programs are written in Wiring (based on Processing), which is similar to the C++ programming language. For Arduino-specific tutorials, check the Arduino section.
  • Codecademy: You can use Codecademy's tutorials to learn multiple programming languages, including Java, HTML/CSS, Javascript, and more.
  • Code.org: Fun tutorials that teach kids the basics of coding using Angry Birds and Plants vs. Zombies.
  • Treehouse: Similar to Codecademy, but with more depth and languages included - tutorials cost money.
  • Udemy: An "online learning marketplace" - collection of code courses, some paid, some free.
  • Scratch: One of the most popular resources for beginners - teaches coding through drag-and-drop tutorials.
  • Code Avengers: Similar to Codecademy, but you get to be called an avenger!
  • Girl Develop It: Helps women find coding workshops and classes in their area.

Hardware and Electrical Engineering

  • SkilledUp Learn Engineering: Collection of links to online courses about electrical engineering.
  • Build Electronic Circuits: List and description of electronic components commonly used in circuits.
  • Beginner's Guide to Electronics: Extensive description of electronic components and how to use them.
  • Learn About Electronics: Collection of articles about various electronics, electricity, and circuits.
  • Sparkfun Breadboard Tutorial: This page covers the basics of breadboards, but links on the button send you to component-specific articles to broaden your knowledge.
  • Fusebox Circuit Builder: Simulate simple circuits with this online tool.
  • Identifying Electronic Components: A guide to help you recognize different components so you know what you're using in your circuits.
  • How to Read a Schematic: Learn how to interpret circuit drawings and diagrams.


About Arduino

  • Arduino Website: Contains general information on how to set up Arduino, as well as reference that covers all included libraries and functions.
  • Arduino 5-Minute Tutorials: Collection of short tutorials on using an Arduino with common robotics components.
  • Lada Ada's Arduino Tutorial: Arduino tutorials to go along with Adafruit's Arduino starter pack.
  • Linux Learn to Program the Arduino: Focuses on the software aspect of Arduino programming.
  • Sparkfun Arduino Tutorial: Introduces beginners to the concept of microcontrollers and explains Arduino basics.
  • Arduino Course for Absolute Beginners: Free 12-part video tutorial system for Arduino learners.
  • The Complete Arduino Guide for Beginners: Starts from the basics and goes through multiple lessons.

Project Ideas

  • Instructables: A variety of user-submitted projects with simple-to-follow instructions.
  • Arduino Playground: Project ideas and directions collected on the Arduino website.
  • Make: Arduino: If you like Make's projects, you can also subscribe to their website for more project ideas.
  • Coduino: Collected projects from various locations. This site also has tutorials about other microcontrollers.
  • Sparkfun Arduino Projects: Sparkfun's projects cover various shields that can be used with Arduino.
  • Pinterest Arduino Projects: Pinterest isn't just for cute crafts and recipes - find Arduino projects here as well.
  • Adafruit Projects: Guides to help you learn about Arduino or create amazing projects.
  • Makerzone Arduino Projects: Projects integrating Matlab and Simulink software.
